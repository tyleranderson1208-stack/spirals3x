## Wipe panel banner (optional)

If you want a featured image on the wipe panel, add this to your `.env` and restart the bot:

```env
WIPEMAP_PANEL_HERO_IMAGE_URL=https://your-cdn-or-discord-image-url
```

You can also change it in Discord with `/wipe-style hero_image:<attachment>` or clear it with `/wipe-style clear_hero:true`.
